# CI4 Docker Premium Starter Kit

Sistema base profesional para desarrollo y despliegue de aplicaciones con CodeIgniter 4, completamente dockerizado y listo para producción.

Diseñado para clonar, configurar y ejecutar en menos de 10 minutos sin decisiones adicionales.

---

## Objetivo real del producto

Este kit elimina toda la configuración manual típica de proyectos PHP:

- infraestructura Docker lista
- entorno CI4 funcional desde el primer arranque
- base para CI/CD real
- preparación para producción desde el día 1

---

## Stack incluido

- CodeIgniter 4.7
- PHP 8.2 (FPM)
- Nginx
- MySQL 8
- Redis
- Supervisor (workers)
- Queue system compatible con Daycry Queue
- GitHub Actions (CI/CD base)
- PHPUnit, PHPStan, Rector
