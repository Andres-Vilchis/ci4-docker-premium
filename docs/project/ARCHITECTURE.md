# CI4 Docker Premium Starter Kit

## Arquitectura oficial

Host:

* Windows 11
* WSL2 Ubuntu

Desarrollo:

* Docker Desktop

Stack:

* Nginx
* PHP-FPM 8.2
* MySQL 8.4
* Redis 7
* Supervisor

Framework:

* CodeIgniter 4.7

Objetivo:
Permitir levantar una aplicación completa mediante:

docker compose up -d

y tener un entorno funcional en menos de 10 minutos.
